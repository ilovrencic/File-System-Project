Trait for inode-like behavior, so that we can have different inodes later on, without having to change the interfaces of the inode trait Solely used in testing, so does not require setter methods

fn get_block(&self, i: u64) -> u64

Get the address of the ith block pointed to by this inode, if there is any block under index i Note that this function's behavior is undefined for unallocated indexes i (as these are commonly set to 0). It is up to the caller to figure out whether the index i was sensible, based on get_size() and the external file system block size.
